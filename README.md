# hackwp

## Usage

**Installation**
```
gh repo clone etragardh/hackwp
cd hackwp
pyinstaller hackwp.py
ln -s dist/hackwp/hackwp.py /usr/local/bin/hackwp
```

**Help**
```
hackwp --help
```
